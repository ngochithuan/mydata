DECOR DREAM WEBSITE - A workspace decor store website
                ------------------

I. Setup instructions:

1. Install Nodejs (If you have not installed)
https://nodejs.org/en/download

2. Install json-server (Using terminal, If you have not installed)
npm install json-server

3. Download the source code from GitHub and extract the ".zip" file
https://github.com/kaliber2602/PC-shop-MID

4. Go to the project folder and install required dependencies (Using terminal):
npm install

5. Run the json-server (Using terminal) to activate the database:
node server.js

6. Then open another terminal in the same directory, Runs the app in the development mode:
npm start

Project will be automatically open at http://localhost:3001
